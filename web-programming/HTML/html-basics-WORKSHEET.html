Question 1

What is the significance of the html element in an html document?

The HTML element contains all other elements on the page. It is known as the root element.
Question 2

What is the significance of the head element in an html document?

This element serves to hold information about the website.
Question 3

What is the significance of the title element in an html document?

This gives your application a name in the browser tab.
Question 4

What is the significance of the body element in an html document?

This is where the content of the website will be.
Question 5

What is the difference between an inline element and a block-level element?

An inline element only fits to the content that exists inside of it. A block-level element takes up the horizontal space.
Question 6

What is a self-closing tag? Also known as an empty element

A tag that cannot hold content and does not need a closing tag.
