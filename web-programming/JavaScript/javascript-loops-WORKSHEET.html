JavaScript Loops Worksheet

Questions

Question 1

What is the difference between a while loop and a for loop?

A while loop will iterate so long as the boolean expression is true. A for loop repeats a code block only a certain number of times.

Question 2

What is an iteration?

Each time the code block executes is called an iteration.

Question 3

What is the meaning of the current element in a loop?

It is the element that is being processed within the body of loop - this changes with each iteration.

Question 4

What is a 'counter variable'?

A variable initialized to a number in the first part of a loop.

Question 5

What does the break; statement do when used inside a loop?

Allows you to terminate the loop early.
