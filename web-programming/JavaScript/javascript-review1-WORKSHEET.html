Question 2

Explain the difference between a local variable and a global variable.

A local variable is defined within a function and is only available to that function while a global variable is defined outside of any functions and is available to all functions and expressions.
Question 3

Which variables in the code sample below are local, and which ones are global? 

var stateTaxRate = 0.06;
var federalTaxRate = 0.11;

function calculateTaxes(wages){
	var totalStateTaxes = wages * stateTaxRate;
	var totalFederalTaxes = wages * federalTaxRate;
	var totalTaxes = totalStateTaxes + totalFederalTaxes;
	return totalTaxes;
}
local: totalStateTaxes, totalFederalTaxes, totalTaxes global: stateTaxRate, federalTaxRate

Question 10

Explain the difference between truthy and falsy values in JavaScript. Provide a few examples of both truthy and falsy values.

A truthy value is any variable that is storing a value; it will evaluate to true if used in a boolean expression. Falsey values are values like 0, null, undefined, NaN, or an empty string that will return as false when used in a boolean expression.

Question 12

Explain the difference between the strict equality operator (===) and the equality operator (==).

The strict equality operator compares the value and the data types of two variables while the equality operator only compares the value of two variables.
Question 13

Explain the difference between stepping over and stepping into a line of code when using the debugger.

Stepping over a line of code will execute the function and move onto the next line of code and stepping into is a line of code will take you to the function that is being invoked.
