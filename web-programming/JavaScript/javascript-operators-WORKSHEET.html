Operators Worksheet

Checkout this page to review the concepts we covered in class regarding operators.

Question 1

What are the comparison operators used for?

They can be used for boolean expressions.

Question 2

Explain the difference between the logical AND operator (&&) and the logical OR operator (||).

The AND operator evaluates if two boolean expressions are both true while the OR operator evaluates two boolean expressions to see if one or the other is true.

Question 3

Which operator would you use to find the remainder from dividing 2 numbers.

The modulus operator "%"

Question 4

Which operator would you use if you wanted to find out if two values were NOT equal?

The inequality operator "!="
